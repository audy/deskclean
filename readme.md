# DeskClean [![Build Status](https://travis-ci.org/audy/deskclean.svg?branch=master)](https://travis-ci.org/audy/deskclean)

Keep your `~/Desktop` tidy!

## Usage

```sh
$ deskclean
```

Run `deskclean -h` for help:

```
Usage of ./deskclean:
  -path string
        path to clean (default "/Users/austin/Desktop")
```
